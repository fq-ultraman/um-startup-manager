## [0.1.1] - 2025-12-31

### 新增

- **延迟执行功能**：支持为启动项设置延迟时间，监控到进程后等待指定秒数再执行操作
- **自动退出功能**：所有最小化任务完成后可自动退出应用（适用于自启动模式）
- **监控状态显示**：状态栏实时显示监控线程运行状态和监控进程数
- **执行时间记录**：记录每次最小化操作的执行时间，便于查看历史记录
- **延迟任务管理**：新增延迟任务队列，支持延迟执行的启动项管理
- **CHANGELOG.md**：创建了变更日志文件，规范版本更新记录

### 改进

- **进程监控系统优化**：
  - 无任务时自动停止监控线程，减少资源占用
  - 支持从监控列表中移除已处理的项目
  - 增强了进程过滤条件，避免重复处理
- **前端界面改进**：
  - 添加了状态栏，显示监控状态和统计信息
  - 优化了统计信息显示位置和样式
  - 启动项设置弹窗新增延迟执行设置
  - 启动项列表显示上次执行时间
- **配置管理增强**：
  - 新增延迟执行时间配置
  - 新增自动退出功能配置
  - 优化了配置保存和加载逻辑
- **响应式设计**：
  - 支持深色模式下的状态栏和执行时间显示
  - 优化了各种状态下的视觉反馈

## 版本管理规范

本项目使用 [语义化版本控制](https://semver.org/lang/zh-CN/)。版本格式为：

**主版本号.次版本号.修订号**

- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你添加了向下兼容的新功能
- **修订号**：当你做了向下兼容的问题修正

## 更新日志格式

每个版本的更新日志应包含以下部分：

- **新增**：新添加的功能
- **改进**：对现有功能的改进
- **修复**：修复的 bug
- **移除**：移除的功能
- **文档**：文档的更新
- **重构**：代码重构
